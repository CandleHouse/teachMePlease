## 项目详解

### 1. [mouse_sensor](./mouse_sensor/) — 光学鼠标传感器模拟

**它做什么？** 用 Python 从零模拟一颗光学鼠标传感器芯片的工作流程：生成桌面表面纹理 → 传感器逐帧拍摄 18×18 图像 → 用频域相位相关法计算帧间位移 → 累加位移还原出鼠标运动轨迹。

**你会学到什么？**

- **离散傅里叶变换（DFT）**：把图像从空间域转换到频域意味着什么
- **频域相位相关**：如何利用 FFT 的平移性质，在 O(N² log N) 时间内检测两幅图像之间的位移
- **Simplex Noise**：比 Perlin Noise 更好的程序化噪声算法，用于生成逼真的表面纹理
- **分形布朗运动（fBm）**：通过多八度噪声叠加，制造多尺度的自然纹理
- **卷积定理**：互相关在频域中的高效计算

### 2. [gesture_recognition](./gesture_recognition/) — 摄像头实时手势识别

**它做什么？** 打开电脑摄像头，实时识别画面里的手势，用 MediaPipe 提取每只手 21 个关键点，再根据「指尖 vs 指根」的几何关系数出伸直的手指（输出阿拉伯数字 `0~5`），并用方框框出手的位置、画出骨架。

**你会学到什么？**

- **两阶段检测流水线（two-stage pipeline）**：为什么先「找手掌」再「标关键点」，而不是让一个网络一步到位
- **卷积层在做什么**：3×3 卷积核如何在图上滑动、为什么局部连接 + 权值共享 + 平移等变让卷积适合做视觉
- **回归 vs 分类**：网络最后一层「输出是哪一类」用分类（交叉熵），「输出是多少」用回归（MSE）；MediaPipe 的 21 个点坐标走的是回归路线
- **深度可分离卷积（MobileNet）**：把普通卷积拆成「逐通道空间卷积 + 1×1 通道混合」，参数量降约 8 倍，这是手机端实时跑的秘诀
- **图像坐标系（y 向下）**：为什么 OpenCV 里 y 轴向下增长，以及它如何决定 `指尖 y < 指根 y → 伸直` 这行判断的符号
- **轴对齐包围盒（AABB）**：画个框框住手，背后只是 `min/max` 几何运算，以及「够用就好」的工程取舍
